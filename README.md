# Cricket Tournament Data Analytics (SQL Project)

This is a simple, beginner-friendly SQL project designed to manage and analyze data for a Cricket Tournament. It helps to track player performances, runs scored, and wickets taken across matches.

## Database Design (Schema)

The project consists of 3 relational tables:

1. **Teams Table:** Stores team details (ID,

Name, City).

2. **Players Table:** Stores player details and links them to their respective teams.

3. **MatchStats Table:** Stores

match-by-match performances of each player (Runs, Balls faced, Wickets taken).

## Key Features & Queries Included:

* Basic database and table creation using Primary and Foreign keys.

* Simple Table Joins to connect players with their team names.

* Data Aggregation using `SUM()` and GROUP BY to find total runs scored by batsmen.

* Data Sorting using 'ORDER BY DESC to find the top wicket-taker of the tournament.
